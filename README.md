# Usage
``` bash
$ cob-t "jordan 1 retro" False
```
- "찾을 신발 이름" jordan 1 retro 혹은 레트로 아닌제품은 닉네임 키워드로 검색가능(ex : black toe)
- 평균리셀가 내림차순으로 볼 경우 Fasle 오름차순으로 볼경우 True 입력
